WEMS Third Party Data API
===

### Getting started

Add the repository to your composer.json:

```
{
    "name": "my/project",
    "repositories": [
        {
            "type": "git",
            "url": "https://github.com/WEMS/third-party-data-client.git"
        }
    ],
    "require": {
        "wems/third-party-data": "0.1.0"
    }
}
```

Run composer install or update.
